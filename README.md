# 💼 Freelancer Tax Calculator (India)

A smart tax calculator built for Indian freelancers, consultants, and professionals to estimate income tax, GST applicability, and get actionable tax-saving insights.

---

## 🚀 Features

- ✅ Income Tax Calculation (Latest Slabs - New Regime)
- ✅ 44ADA Presumptive Taxation Support
- ✅ Expense-based Profit Calculation
- ✅ GST Applicability Check (₹20L threshold)
- ✅ TDS Adjustment
- ✅ Advance Tax Alerts
- ✅ Smart Tax Saving Suggestions
- ✅ Lead Capture Form (for CA consultation)

---

## 🧮 What This Tool Solves

Freelancers often:
- Overpay taxes
- Ignore 44ADA benefits
- Miss GST compliance
- Don’t plan advance tax

👉 This tool gives a **quick snapshot + optimization suggestions**

## 🧠 Logic Overview

### Income Calculation

* 44ADA → 50% profit assumed
* Otherwise → Expense-based calculation

### Tax Calculation (New Regime)

* Slab-based system
* Rebate u/s 87A applied
* 4% cess included

### GST Rule

* Mandatory if turnover > ₹20 Lakhs
